3. Instruções para Teste Manual

  1. Deletar banco atual: Execute no MySQL: DROP DATABASE sgqoti;

  3. Acessar sistema: As migrations vão recriar tudo automaticamente
  4. Ir para Amostragens: Menu → Amostragens
  5. Criar nova amostragem:
    • Preencher número da NF
    • Selecionar responsáveis
    • Anexar PDF (será salvo em MEDIUMBLOB)
    • Se status = reprovado, anexar fotos (MEDIUMBLOB)
  6. Verificar: PDF e fotos devem ser servidos diretamente do banco

5. Comandos SQL para Verificação

-- Verificar estrutura da tabela amostragens
DESCRIBE amostragens;

-- Verificar estrutura da tabela amostragens_evidencias  
DESCRIBE amostragens_evidencias;

-- Ver amostragens com tamanho dos BLOBs
SELECT id, numero_nf, status, 
       LENGTH(arquivo_nf_blob) as pdf_size,
       arquivo_nf_name,
       (SELECT COUNT(*) FROM amostragens_evidencias WHERE amostragem_id = a.id) as evidencias_count
FROM amostragens a;

-- Ver evidências com tamanho das imagens
SELECT id, amostragem_id, name, type, LENGTH(image) as image_size 
FROM amostragens_evidencias;
